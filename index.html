          Frequently asked questions

          What is this?

          This is a website to lookup games without age rate for Steam, built on GitHub pages

          Are unrated games a problem?

          Because of a new regulation in Germany, all games without an age rating are no longer allowed to be sold in Germany

          So, this includes which games exactly?

          Sadly, this will include all games without an age rating

          Isn't age rating via official organizations either slow or costly?

          This does not matter because steam itself provides a solution that is free and can be done via a form on the steam partner page

          When will this be a problem?

          On November 15th all games without an age rating will be delisted from the Steam Store in Germany

          What about games I've already bought?

          If you purchased a game and it does not have an age rating, you don't have to worry and you can still play the game after the deadline

          Can newer games be uploaded without age rating?

          No. As of right now, the upload process for Steam requires age verification

          What can I do if found a game which is not rated?

          First off, you should double-check the official Steam Store page for the game to ensure that the data of this site is not outdated

          If it seems that there is no rating, then you can only rely on the game's developer to initialize the rating process

          Please do not pressure developers who don't already have or do not want to rate their games

          For more info, lookup the official statement and FAQ from steam

          Statement: https://steamcommunity.com/groups/steamworks/announcements/detail/4678768276768588864

          FAQ: https://partner.steamgames.com/doc/gettingstarted/contentsurvey/germany

          Personal thoughts:

          It's quite sad that many games will be delisted because of this

          The preferred solution should be that every unlisted game should be listed as 18+ if no rating has been given

          However, this idea has already been rejected by the government

          This affects mainly games from smaller studios, which no longer maintain or are sometimes unable to update games due to various reasons

          In the end, this results in some great indie games that can no longer be purchased

          Don't be discouraged or enraged because some games will be unlisted

          There's always the possibility for Steam or Germany to handle this situation differently in the future, so that we can buy unrated games again
